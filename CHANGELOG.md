# Changelog

All notable changes to `notyfyre` will be documented in this file.

## [1.0.3] - 2025-07-14

### Added

- Initial release
- Basic toast notification functionality
- Support for Success, Error, Warning, and Info notifications
- PHP and JavaScript API
- Laravel integration with Facade and Service Provider
- Blade components and directives
- Configurable options (position, timing, styling)
- Auto-handling of Laravel flash messages
- Responsive design
- Progress bar support
- Custom styling options
- Helper functions and traits
- Middleware for automatic error handling
- Base64 encoded icons (no external dependencies)

### Features

- ✅ PHP 8.0+ support
- ✅ Laravel 9.0+ support
- ✅ Responsive notifications
- ✅ Multiple positioning options
- ✅ Customizable styling
- ✅ Auto-hide with configurable timing
- ✅ Close button support
- ✅ Progress bar indicator
- ✅ No external dependencies
- ✅ Easy integration
